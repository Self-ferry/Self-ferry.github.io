# 总结：

为什么要分成 3 张表？

1、函数导出的个数与函数名的个数未必一样。所以要将函数地址表和函数名称表分开.

2、函数地址表是不是一定大于函数名称表？

未必，一个相同的函数地址，可能有多个不同的名字.
